About

Python rewrite by Bek kobro, customized from the original PHP web UI for modern Raspberry Pi OS.

Python Flask Docker lgpio

Origin

Forked from timkn/Raspberry-Pi-GPIO-Control by TK (2020) — a PHP + WiringPi project for browser-based GPIO control.

This version

Flask serves the UI; Python drives GPIO via WiringPi CLI or lgpio. jQuery and Ajax handle live pin state updates from any device on your LAN.
